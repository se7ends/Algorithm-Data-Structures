# Patika.dev Insertion Sort Projesi

Verilen dizi: [22, 27, 16, 2, 18, 6]

Insertion Sort mantığına göre sıralama aşamaları:
1. [22, 27, 16, 2, 18, 6] (27 incelenir, yeri doğru)
2. [16, 22, 27, 2, 18, 6] (16 başa gelir)
3. [2, 16, 22, 27, 18, 6] (2 en başa gelir)
4. [2, 16, 18, 22, 27, 6] (18 araya girer)
5. [2, 6, 16, 18, 22, 27] (6 araya girer ve sıralama biter)

Big-O Gösterimi: O(n^2)

Time Complexity (18 Sayısı)
Dizi sıralandıktan sonra: [2, 6, 16, 18, 22, 27]
Aranan sayı "18", dizinin tam ortasında yer aldığı için 'Average Case' kapsamına girer.
